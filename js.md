### Errors
1. TypeError;
- TypeError: Assignment to constant variable.
- TypeError: Cannot read properties of undefined (reading 'toUpperCase');
> при обращении метода / свойства к пустой строке возникает такая ошибка.
2. SyntaxError;
- SyntaxError: Missing initializer in const declaration;
- SyntaxError: missing ) after argument list;
  
> Переменные необходимы только при работе с циклами.
> В остальном можно обойтись без них, используя константы.

### Интерполяция
Вставка переменных и выражений в строку для получения динамического текста.
` - символ бектик.
> шаблонные литералы - синтаксические конструкции для выполнения интерполяциию.
1. Обратные кавычки, бектики;
2. ${name} - переменные;
3. ``` line1 line2 line3 ``` - моногострочный вывод; 

### JS язык со слабой типизацией
> JavaScript был создан для интернета, а в интернете вся информация — это строки.
> Даже когда вы вводите на сайте номер телефона или год рождения, на сервер эта информация поступает не как числа, а как строки.
> Поэтому авторы языка решили, что автоматически преобразовывать типы — правильно и удобно.

### Boxing
Переменные с примитивным типом данных преобразуются в объект при обращении к ним.
```
const str = 'string';
str.toLowerCase();
// str - объект
```

### || &&
`||`
> Оператор ИЛИ работает так, что его выполнение (слева направо) прерывается 
> и возвращается результат первого аргумента, который можно преобразовать в true.
* если ничего не найдено, возвращает последний операнд.

`&&`
> Оператор И работает так, что его выполнение (слева направо) прерывается
> и возвращается результат первого аргумента, который можно преобразовать в false.
* если ничего не найдено, возвращает последний операнд.

**0, '', undefined, NaN, null приводятся к false. Эти значения называют falsy.**

### Агрегация данных (числа)
> Задачи в которых нужно перебрать весь набор данных, т.е. цикл.
> Поиск мин/макс значения, среднего арифметического, суммы например.

* Нейтральный элемент операции - число, которое не изменяет значение при операции.
*Примеры нейтральный элементов*
```
'' + 'one' // пустая строка в конкатенации
1 * num // 1 в умножении
```

### Агрегация данных (строки)
> Это такие задачи, в которых строка формируется динамически, 
> то есть заранее неизвестно, какого она размера и что будет содержать.
*Пример*
```
('str').repeat(times) // метод использует цикл для динамического создания строки
```
