### Forms
attributes
1. `size` - width text field
2. `list` for `<input>` element
1. create list of hints
```
<datalist id="colors">
    <option value="red">red</option>
    <option value="blue">blue</option>
    <option value="green">green</option>
  </datalist> 
```
2. connect with `input` elemnent
```
<form>
<input type="text" list="color" name="color">
</form>
```
3. `name`	Имя поля, для того, чтобы обработчик формы мог его идентифицировать.
4. `disabled`	Блокирует поле для ввода текста и не отправляет данные на сервер.
5. `form`	Идентификатор формы для связывания текстового поля с элементом <form>.
6. `type`	Для текстового поля мы указываем значение text или вообще не пишем этот атрибут.
7. `maxlength`	Устанавливает максимальное число символов, которое может быть введено пользователем в текстовом поле.
8. `readonly`	Блокирует поле для ввода текста.
9. `value`	Начальный текст, отображаемый в поле.
10 `autocomplete`	Введённый ранее текст запоминается браузером и подставляется при следующем вводе.
11. `autofocus`	Поле получает фокус после загрузки документа.
12. `pattern`	Шаблон ввода текста.
13. `required` Указывает, что поле является обязательным для заполнения.
14. `placeholder`	Добавляет подсказку, которая исчезает при вводе текста.
15. `dirname` Имя переменной, которая отправляется на сервер и автоматически получает значение ltr или rtl

#### Атрибуты для элемента `<textarea>`
1. `rows` Высота поля в строках текста.
2. `cols` Ширина поля в символах.
3. `wrap` Значение soft передаёт на сервер текст одной строкой, а hard учитывает значение cols и автоматически добавляет переносы.

#### Атрибуты для элемента <input type="radio/checkbox">
1. `checked` Заранее выбранный

#### Элемент для загрузки файлов 
`<input type="file">`
> Его атрибуты
1. `name`	Имя поля; используется для его идентификации обработчиком формы.
2. `disabled`	Блокирует поле для отправки файлов.
3. `form`	Идентификатор формы для связывания поля с элементом <form>.
4. `type`	Для загрузки файлов значение должно быть file.
5. `accept`	Устанавливает фильтр на типы файлов, которые вы можете открыть через поле загрузки файлов.
6. `autofocus`	Поле получает фокус после загрузки документа.
7. `required`	Указывает, что поле является обязательным для заполнения.
8. `multiple`	Позволяет выбирать и загружать сразу несколько файлов.

> [!IMPORTANT]
> Для отправки файлов в форме необходимо сделать следующее:
> 1. задать метод отправки данных POST `method="post"`
> 2. установить у input атрибут `enctype="multipart/form-data"`.

#### File через атрибут `accept` можно ограничить тип файлов
1. `image/jpeg`	Только файлы в формате JPEG.
2. `image/jpeg,image/png` Только файлы в формате JPEG и PNG.
3. `image/*` Любые графические файлы.
4. `audio/*` Любые аудио файлы.
5. `image/*,video/*` Любые графические и видеофайлы.
6. `application/pdf` ПДФ файл

*<input type="file" name="photo" multiple accept="image/*,image/jpeg">*

#### Раскрывающийся список `<select>`
**Атрибуты <select>**
1. `name`	Имя списка для идентификации поля.
2. `disabled`	Блокирует список для выбора.
3. `form`	Идентификатор формы для связывания поля с элементом <form>.
4. `size`	Число пунктов списка, показываемых пользователю.
5. `multiple`	Позволяет выбрать несколько пунктов из списка.
6. `autofocus`	Элемент получает фокус после загрузки документа.
7. `required`	Перед отправкой формы обязательно следует выбрать пункт списка.

**Атрибуты <option>**
1. `disabled`	Блокирует пункт списка для выбора.
2. `selected`	Позволяет заранее выбрать пункт списка.
3. `label`	Текст, который отображается в теге группировки списка.
4. `value`	Значение выбранного пункта списка, отправляемого на сервер.
*Пример разметки*
```
<form>
   <select name="day">
    <option value="mon">Понедельник</option>
    <option value="tue">Вторник</option>
    <option value="wed" selected>Среда</option>
    <option value="thu">Четверг</option>
    <option value="fri">Пятница</option>
    <option value="sat">Суббота</option>
    <option value="sun">Воскресенье</option>
   </select>
  </form>
```
*Пример группировки списка*
```
<form>
   <p><select name="food">
    <optgroup label="Русская кухня">
     <option value="r1">Закуска Барская</option>
     <option value="r2">Раки, фаршированные по-царски</option>
     <option value="r3">Биточки в горшочке</option>
    </optgroup>
    <optgroup label="Украинская кухня">
     <option value="u1">Галушки славянские</option>
     <option value="u2">Пампушки украинские</option>
     <option value="u3">Жаркое по-харьковски</option>
    </optgroup>
   </select></p>
   <p><input type="submit" value="Отправить"></p>
  </form>
```
