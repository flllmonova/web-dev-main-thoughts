### Position
1. ` static `
2. ` relative `
3. ` absolute `
4. ` fixed `
5. ` sticky `
*z-index работает только для элементов, у которых position задано как relative, absolute, fixed или sticky*

### Как сдвигаются элементы
```
element {
  position: relative;
  top: 5px; /* от верха вниз на 5px */
  top: -5px; /* вверх на 5px */

  left: 5px; /* от левой стороны 5px */
  left: -5px; /* влево 5px */

  right: 5px; /* от правой стороны 5px */
  right: -5px; /* вправо 5px */

  bottom: 5px; /* от низа 5px */
  bottom: -5px; /* вниз 5px */
}
```

#### Relative position
* смещение относительно своей позиции в разметке;
* может быть точкой отсчёта для дочернего элемента с `position: absolute`.

#### Анимации с `position: relative;`

```
/* при наведении на картинку она поднимается и увеличивается */
img {
    border: 1px solid #ce242b; /* Параметры рамки */
    position: relative; /* Относительное позиционирование */
    transition: 1s; /* Время анимации */
}

img:hover {
    z-index: 10; /* Поверх всех картинок */
    transform: scale(1.2); /* Масштабирование на 20% */
}
```

```
/* при наведении кнопка закрывает свою тень */
button {
    background: #CE242B; /* Красный фон */
    color: #fff; /* Белый цвет текста */
    border: none; /* Убираем рамку */
    padding: .7rem 2rem; /* Поля вокруг текста */
    box-shadow: 5px 5px 0 #000; /* Параметры тени */
    position: relative; /* Относительное позиционирование */
    left: 0; /* Положение слева */
    top: 0; /* Положение сверху */
    transition: 1s; /* Время анимации */
}

button:hover {
    left: 5px; /* Положение слева */
    top: 5px; /* Положение сверху */
    box-shadow: 0 0 0 #000; /* Параметры тени */
}
```

### Position Fixed
- Позиционируется относительно окна браузера;
- Если занулить параллельные стороны (left + right / top + bottom), то элемент будет занимать весь inlise-size / block-size;
- Если занулить все стороны -> всю площадь.

### Position Sticky 
* Обязательно указать 1 сторону для привязки;
* Не работает для вложенного элемента.

### Box-model
#### background-clip 
свойство определяющее в каких областях элемента будут отображаться фоновая картинка и фоновый цвет.
`background-clip: border-box | padding-box | content-box | text;`
> [!IMPORTANT]
> Делать цвет `transparent` чтобы background-clip сработал

### Margin Border Padding
* Отрицательный `margin` возможен, отрицательный `padding` и `border` - нет

#### Margin
* Выравнивание блока по центру через значение `margin: auto;` работает только в сочетании с `width`.
* В качестве значения `margin` можно использовать процентную запись, с которой связаны следующие особенности.
> По горизонтали и вертикали проценты считаются от ширины всего блока.
> Схлопывание - объединение двух вертикальных margin в один.
* Схлопывание происходит только у вертикальных `margin`, 2 `margin` комбинируются в один.

Margin не работает для элементов: строчных, строчно-блочных, обтекаемых, flex-элементов, элементов с position absolute / fixed.

### Width & Height
1. Элемент `display: block;`
```
width: 100%; height: auto;
```
2. Элемент `display: inline;`
```
width: auto; height: auto;
```

### Блочные элементы
* Блочные элементы это `display: block / list-item / table;`
* Свойство vertical-align не применимо к блочному элементу.

### Строчные элементы
* `margin-inline` работает, `margin-block` не применим;
* border & padding работают;

### Строчно-блочные элементы
* Ширина и высота блока получается сложением значений width/height, margin, border и padding.
* Элементы можно выравнивать по вертикали с помощью свойства vertical-align.
* Несколько элементов идущих подряд располагаются на одной строке и переносятся на другую строку при необходимости.
* У `button`, `input` и `textarea` по умолчанию `display: inline-block;`.
