#### Селекторы
1. Перечисление селекторов
```
q, p { }
// и тег p и тег q
```
2. Объединение селекторов
```
em.date {
  color: blue;
}
```
3. Иерархия
```
header a {
  color: red;
}
```
4. Псевдоклассы привязаны к обычным селекторам и начинаются с двоеточия.
```
a {
  color: blue;
}
  
a:hover {
  color: red;
}
```
